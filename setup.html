Ersteinrichtung

Nova richtet PocketBase automatisch im Docker-Container ein.

1. Allgemeine Informationen

Dieser Name wird in der App und in E-Mails angezeigt.
Optional können Sie direkt in der Ersteinrichtung ein eigenes SVG-Logo speichern.

2. Super-Admin-Konto erstellen

Dieser Benutzer wird automatisch als Administrator (Super-Admin) angelegt.

3. E-Mail Einstellungen (Optional)

Diese Einstellungen werden für Benachrichtigungen benötigt. Sie können diesen Schritt überspringen.

Bei Gmail wird dringend ein App-Passwort empfohlen.
